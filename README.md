# Mapage Client NextJS

A web client for [Mapage](https://crates.io/crates/mapage) intended to support all of its possible configurations.

Supports Mapage 0.2.0 

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.

## How To Connect To A Server

1. Make sure your Mapage instance is running.
2. Make sure Mapage-client-nextjs is also running.
3. Click on "Connect To Server" in the left side menu.
4. In the "URL" field input the server address e.g. http://localhost:8000/.
5. Click the button labeled "CONNECT".
6. Done!
